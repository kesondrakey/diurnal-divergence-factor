# diurnal-divergence-factor

Code and data for analyses supporting the manuscript, "Towards Biophysically-informed Drought Indicators Across US Ecosystems".

## Repository contents

### data
Input data used for analyses and figure generation.

- `Masterfile.csv`: processed analysis dataset used for statistical analyses and most figures
- `US-SRS.csv`: site-level data used for Figure 1

### scripts
R Markdown documents and supporting scripts used to reproduce analyses and figures.

- `Statistics.Rmd`: statistical analyses reported in the manuscript
- `Statistics.pdf`: rendered output of `Statistics.Rmd`
- `Figures.Rmd`: figure generation workflow
- `Figures.pdf`: rendered output of `Figures.Rmd`
- `Figure_2_REQUIRES_EXTERNAL_FILE.R`: script used to generate Figure 2; requires an external aridity index raster file not included in this repository because of file size

### figures
Output figures generated by `Figures.Rmd`

### tables
Output tables generated by `Statistics.Rmd`

## Reproducibility

The main manuscript analyses can be reproduced using the files in this repository. `Masterfile.csv` is the primary processed dataset used for the statistical analyses. Render `Statistics.Rmd` to reproduce tables and statistical outputs, and render `Figures.Rmd` to reproduce manuscript figures.

## External data required for Figure 2

Figure 2 requires an external aridity index raster (`.tif`) that is not included in this repository because of file size limitations. To reproduce Figure 2, download the Global Aridity Index raster from the source below and update the file path in `scripts/Figure_2_REQUIRES_EXTERNAL_FILE.R`.

**Source**  
Zomer, R. J., Xu, J., and Trabuco, A. (2022). Version 3 of the Global Aridity Index and Potential Evapotranspiration Database. *Scientific Data*, 9, 409.

**Access link**
- https://figshare.com/articles/dataset/Global_Aridity_Index_and_Potential_Evapotranspiration_ET0_Climate_Database_v2/7504448/6
